# 303 Discord Bot

## Bot Nasıl Başlatılır?

### Yöntem 1: Batch Dosyası (Kolay)
`start_bot.bat` dosyasına çift tıklayın.

### Yöntem 2: PowerShell
`start_bot.ps1` dosyasına sağ tıklayıp "PowerShell ile çalıştır" seçin.

### Yöntem 3: Manuel
1. Komut satırını açın
2. `cd bot` yazın
3. `python main.py` yazın

## Gereksinimler
- Python 3.8+
- discord.py
- python-dotenv

## Bot Özellikleri
- Ticket sistemi
- İstatistikler
- Unban komutu
- Restart komutu
- Çekiliş sistemi
- Clear komutu

Bot başarıyla başladığında "303BOT#1425 olarak giriş yapıldı!" mesajını göreceksiniz.
